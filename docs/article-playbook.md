# Article Playbook

Use this as the default standard for every new article on both sites.

## Main Site Brief

Every main-site post should:

1. Target one primary keyword only.
2. Answer the query in the first 80 words.
3. Sound like a calm mechanic talking to someone stranded, not a content marketer.
4. Give a clear next action: keep driving, stop driving, call for help, or inspect safely.
5. Include a short cost section when the question has payment intent.
6. Include safety bullets when the user may be roadside.

Main-site structure:

1. Title that closely matches the query
2. Meta description under 155 characters
3. `Quick Answer`
4. `What To Do`
5. `What It Might Cost` when relevant
6. `Stay Safe` when relevant
7. Internal links to related breakdown, insurance, or towing pages

## Feeder Brief

Every feeder post should:

1. Focus on a cost, pricing, comparison, or vehicle-type angle.
2. Support the main site instead of competing with it.
3. Use realistic pricing ranges and explain what changes the quote.
4. Include a clean canonical when there is a matching main-site target.
5. End with a natural backlink to Tow With The Flow.

## Publish Checklist

Before publishing:

1. Make sure the title, slug, and H1 all match the target keyword closely.
2. Confirm the intro answers the searcher without fluff.
3. Add 3 to 5 internal links.
4. Add one image only if it helps the page feel complete.
5. Check the meta description for clarity and urgency.
6. Confirm schema, canonical, and robots behavior render correctly in the built page.
7. Make sure the page is better than the current top results on direct usefulness, not just length.

## Anti-Slop Rules

Do not publish articles that:

1. Exist only because a random modifier was appended to a better keyword.
2. Repeat a page you already have with only minor wording changes.
3. Spend too much time on generic car education before telling the reader what to do.
4. Hide the real answer behind filler intros.
5. Target a topic with weak urgency and weak monetization before stronger pages are done.
