# WordPress SEO + Theme Kit

This is the practical setup for `towwiththeflowyo.wordpress.com` if the goal is:

- easy maintenance
- clean SEO
- fast loading
- obvious article structure
- no spammy look

## Recommended Theme

### Best default pick: Twenty Twenty-Five

Use `Twenty Twenty-Five` if you want the easiest, safest setup.

Why:

- built by the WordPress team
- simple and adaptable
- includes patterns for homepage / landing page layouts
- strong accessibility baseline
- good fit for article-heavy sites

### Alternate pick: Assembler

Use `Assembler` if you want a more minimal, cleaner editorial feel with less visual clutter.

Why:

- simple and minimal
- full-site editing support
- clean blog / professional / modern positioning

## Recommendation

Use `Twenty Twenty-Five` first.

It is the better "easy peasy" choice for:

- homepage sections
- blog archive clarity
- mobile readability
- future edits without theme fighting

## Tag Rules

Keep tags tight. Do not create endless one-off tags.

Use tags for:

- symptom
- location
- condition
- service type

Examples:

- `dead battery`
- `engine overheating`
- `portland oregon`
- `after hours`
- `rainy weather`

## Internal Linking Rules

Each WordPress post should ideally point readers toward:

- one closely related symptom article
- one related towing-cost article
- the main Tow With The Flow canonical site

Suggested reusable related-post blocks:

- battery / alternator / starts-then-dies cluster
- overheating / smoke / fluid leak cluster
- highway blowout / bridge breakdown / shoulder safety cluster
- city towing cost clusters

## Visual Style

Keep it simple:

- white background
- dark text
- one accent color only
- featured image on posts
- readable paragraph width
- no busy sidebar
- no popups
- no slider

## WordPress Setup Checklist

1. Activate `Twenty Twenty-Five`.
2. Create a static page called `Home`.
3. Create a posts page called `Roadside Guides`.
4. Set `Settings > Reading`:
   - Homepage: `Home`
   - Posts page: `Roadside Guides`
5. Add the homepage copy from this file.
6. Create the top nav links listed above.
7. Keep one clean footer nav.
8. Use categories from this file only.
9. Do not add plugin bloat unless there is a clear reason.
